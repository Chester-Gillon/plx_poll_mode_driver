== Description ==

This describes how to compile GscApi under AlmaLinux 8.6 with a 4.18.0-372.26.1.el8_6.x86_64 Kernel.

[mr_halfword@haswell-alma ~]$ cd ~
[mr_halfword@haswell-alma ~]$ tar -xzf ~/Downloads/GscApi-1.6.6.tar.gz

[mr_halfword@haswell-alma ~]$ cd ~/GscApi-1.6.6
[mr_halfword@haswell-alma GscApi-1.6.6]$ mkdir buildGsc
